# Icon 图标

vueTensils 提供了一套常用的图标集合。
如果你想像用例一样直接使用，你需要全局注册组件，才能够直接在项目里使用。


```md
<script setup>
import { useData } from 'vitepress'

const { theme, page, frontmatter } = useData()
</script>

## Results

### Theme Data
<pre>{{ theme }}</pre>

### Page Data
<pre>{{ page }}</pre>

### Page Frontmatter
<pre>{{ frontmatter }}</pre>
```
